My Hero Academia | The lord of Alter

Projet profesionnel : Conception Jeux-Vidéo et Site-Web.
Dans le cardre du projet d'un stage avec "Bones".
Idée de conception du projet :
- "Gacha Game" : Les jeux Gacha sont des jeux vidéo qui implémentent le mécanisme Gacha. Il est similaire à la boîte à butin, car il incite les joueurs à dépenser de l'argent virtuel ou réel dans le jeu pour recevoir un objet virtuel aléatoire.
Langage utilisé : C++ avec SFML comme interface graphique.

- "Site-Web" : reprenant le jeu comme thème
Langage utilisé : HTML/CSS/JS/angularJS.
